# handson-about-setup-js-env

社内勉強会資料（モダンな JS をするための環境構築講座）

`SLIDE.md` が 発表資料. `reveal.js`でスライド化される.

webpack で hello world するまではライブコーディングします。
その続きは[こちら](starter)をみておいてください。
